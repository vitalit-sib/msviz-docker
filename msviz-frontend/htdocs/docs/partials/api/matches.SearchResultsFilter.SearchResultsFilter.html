SearchResultsFilter
object in module matches

Description

It shall define all what is available from a view page.

A bean containing all what is to restrict a search results:

  • searches
  • modification
  • proteinAC

It will contain selected values, but also the selected one. And selecting a few searches will have an effect on available modifications and then protein This will be some "active" bean, as setting one will trigger the available other and so on
